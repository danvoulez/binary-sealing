# Glossary

- **Act**: consequence-bearing vector (nine slots)
- **Dynamic Projection**: bounded reasoning surface over the Act field
- **Diamond**: non-collapsing span; sealed stable projection
- **Enzyme**: signed, reproducible transform
- **Lens**: projection contract defining how to slice the Act field
- **Operator**: bounded agent in a specific territory
- **Receipt**: record emitted by every external effect
- **Vault**: custody chamber for payloads and Diamond artifacts
