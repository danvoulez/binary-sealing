## Invariants

1. Admission is not activation.
2. A projection is a bounded reasoning surface, not truth itself.
3. Doubt must never collapse silently into confidence.
4. A Diamond preserves provenance, consequence, uncertainty, transform history, and custody across observation.
5. Every external effect emits a receipt.
6. If no safe anchor exists, the agent must declare insufficient grounding.
